Before You Start

  • Unlocked bootloader is mandatory
  • Latest official firmware must be installed
  • OrangeFox recovery image should be downloaded

Reboot your device into fastboot mode:

adb reboot fastboot

Verify that the device is detected:

fastboot devices

Boot OrangeFox recovery:

fastboot boot orangefox.img

In recovery, format data and start ADB sideload. Then run:

adb sideload rom.zip

Note: If the progress stops at 47%, do not panic. This is normal and the installation will continue automatically.
