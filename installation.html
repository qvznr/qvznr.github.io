Installation Guide

Prerequisites

  • Unlocked bootloader + latest Official update installed
  • Download OrangeFox recovery

Recovery in Fastboot Mode

  1. Reboot your device into fastboot with: adb reboot fastboot
  2. Verify your device is detected: fastboot devices
  3. If detected, boot into OrangeFox recovery:
    • fastboot boot orangefox.img

Sideload ROM

  1. Go to recovery: adb reboot recovery or fastboot reboot recovery
  2. Format Data in recovery menu
  3. Select ADB Sideload from recovery
  4. Transfer ROM: adb sideload rom.zip
  5. Note: If transfer stops at 47%, this is normal — installation continues automatically

For device-specific details, refer to each ROM's release notes.
