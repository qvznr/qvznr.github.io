Get Started

A short, practical guide to begin using Python.

1. Install Python

Download and install the latest Python 3 from python.org. Make sure to enable Add to PATH on Windows.

2. Verify installation

Open PowerShell (Windows) or Terminal (macOS/Linux) and run:

python --version
3. Run the REPL

Start an interactive prompt by running python, then try:

>> print("Hello, Python!")
Hello, Python!
4. Create your first script

In a new file named hello.py add:

def main():
    print("Hello from a script")

if __name__ == "__main__":
    main()

Run it with python hello.py.

5. Learn with examples

Return to Python Notes for concise examples, or explore resources like the official tutorial: docs.python.org/tutorial.
